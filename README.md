# Filmku App

Aplikasi ini ialah aplikasi yang berguna untuk mencari film, terkhusus untuk film yang bergenre superhero.

## Implementasi

-  React JS
-  Reusable Component
-  Fetch API dari : http://www.omdbapi.com
-  React Routing Auth
-  Eslint
-  Responsive Design

## Cara menjalankannya
1. npm install
2. npm start
